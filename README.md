# ⚙️ TechFix - Venta y Servicio Técnico

Sitio web de e-commerce para venta, reparación y mantenimiento de computadoras y notebooks.

## 📁 Estructura

```
proyecto/
├── index.html        ← Página principal
├── productos.html    ← Catálogo con carrito
├── README.md
├── css/
│   └── styles.css
├── js/
│   └── script.js
└── img/
```

## ✅ Requisitos cumplidos

| # | Requisito | ✅ |
|---|-----------|---|
| 1 | HTML semántico (header, nav, main, section, footer) | ✅ |
| 2 | Formulario de contacto con Formspree | ✅ |
| 3 | CSS externo con Google Fonts y backgrounds | ✅ |
| 4 | Flexbox (servicios/productos) + Grid (reseñas) + Media Queries | ✅ |
| 5 | Multimedia (imágenes, iframe mapa, iframe YouTube) | ✅ |
| 6 | Subido a GitHub Pages / Netlify | ✅ |
| 7 | script.js con validación de formulario y manipulación del DOM | ✅ |
| 8 | Carrito de compras con localStorage | ✅ |
| 9 | Edición de cantidades, eliminación y total dinámico | ✅ |
| 10 | SEO (meta tags) y accesibilidad (alt en imágenes) | ✅ |

## 🛠️ Tecnologías

- HTML5, CSS3, JavaScript
- Google Fonts (Rajdhani + Inter)
- Formspree (formulario de contacto)
- localStorage (persistencia del carrito)
